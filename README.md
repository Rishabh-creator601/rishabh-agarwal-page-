# Rishabh Agarwal — AI / ML Engineer

Personal portfolio site for Rishabh Agarwal, an AI/ML engineer specializing in
computer vision and NLP — from notebook to production: detection pipelines,
sentiment systems, and full-stack ML apps.

**Live site:** https://github.com/Rishabh-creator601/rishabh-agarwal-page-

## Overview

A single-page, dependency-free portfolio built with plain HTML, CSS, and
JavaScript. All content (projects, skills, roadmaps, and links) is driven by a
single data file, so the page rebuilds itself from data without touching the
markup.

## Featured projects

| Project | Focus | Stack |

| Mask Count | Real-time face-mask detection and counting | Python, OpenCV |

## Tech stack

- **Languages:** Python, JavaScript, C / C++, HTML / CSS, Java
- **ML / DL:** TensorFlow, Keras, Scikit-learn
- **Computer Vision:** OpenCV, YOLOv11, MediaPipe, DeepFace, FaceNet
- **NLP:** spaCy, NLTK, HuggingFace Transformers, Gensim
- **MLOps & APIs:** FastAPI, Pydantic
- **Web & Deploy:** Streamlit, Flask, Next.js, Vercel

## Project structure

```
index.html      # The portfolio page (markup + styles)
data.js         # Site content: projects, skills, roadmaps, links
assets/         # Portrait, resume, and project demo videos/gifs
```

## Running locally

It's a static site — open `index.html` in a browser, or serve the folder:

```bash
python -m http.server 8000
# then visit http://localhost:8000
```
